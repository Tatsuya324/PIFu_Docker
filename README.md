# PIFu_Docker

## セットアップ

### 1. 新たにターミナルを立ち上げて次のコマンドでこのリポジトリをクローン（ダウンロード）してください
```sh
git clone https://github.com/TatsuyaOkazaki324/PIFu_Docker.git
```

### 2. 次のコマンドでsetupしてください。
```sh
./setup.sh
```

### Training の方法
```
PIFu_Docker
 |-shared_dir
    |-raw_data
    |   |-raw__100k.obj
    |   |-tex
    |       |-raw__dif_jpg
    |
    |-train_data
```
上記ファイル構成にしなければならない
